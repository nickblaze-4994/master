Personal AI Projects

AI Workflow Orchestration Projects

A single page for my AI workflow prototypes deployed on Vercel. These projects explore agent workflows, guided intake, operational routing, support automation, and lightweight AI product interfaces.

Background

These projects are built around a common theme: using AI to turn messy business inputs into structured next steps. Each prototype focuses on a different workflow, from customer support and vendor operations to guided intake and product-style interfaces.

Support Automation

Woot Support Agent

A customer support workflow prototype for Woot-style support scenarios. The agent handles user questions, checks the relevant policy path, and shows the decision process through an activity panel.

The project demonstrates an LLM tool-calling loop combined with deterministic policy checks, so the user can see both the answer and the workflow behind it.

Open live project

Vendor Operations

Vendor Ops Agent

A vendor operations workflow prototype focused on routing, triage, and operational handoffs. It is designed around the kind of repetitive vendor-facing issues that usually require manual review.

The project shows how AI can sit between incoming requests and the next operational step, helping structure the work before a human reviews it.

Open live project

Guided Intake

Guided Intake

A guided intake workflow that turns user inputs into structured classifications and routing decisions. It is built around the idea that intake forms can be more useful when AI helps interpret context instead of only collecting fields.

The project demonstrates how a queue of intake items can be processed, classified, and routed in a clear interface.

Open live project

AI Product Interface

Portfolio Product

A product-style prototype deployed on Vercel. This project focuses on presenting AI-generated or AI-assisted outputs through a simple user interface.

The goal is to test how lightweight front-end experiences can make AI workflows easier to understand, review, and share.
